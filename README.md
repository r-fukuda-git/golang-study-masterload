# Golang学習リポジトリ

## 概要

本リポジトリは、インフラエンジニアとしてbashシェルスクリプトに慣れ親しんできた筆者が、
Go（Golang）というプログラミング言語の習得に向けて取り組んできた学習記録をまとめたものです。

これまでコマンドライン操作や自動化スクリプトの作成においてbashを使用してきましたが、
より構造化されたプログラムの作成、並行処理の実装、そして型安全性を備えた堅牢なコードを書くことを目標に、Go言語の学習を進めています。

## 学習アプローチ

### Bashとの比較による理解促進

インフラエンジニアとしての経験を活かし、bashで行っていた操作をGo言語でどのように実装するかを意識しながら学習を進めています。
現時点でbashで行っていた操作をGo言語では下記のように理解しつつ学習を進めています。

- **ファイル操作**: `cat`, `grep`などのコマンドの動作をGo言語で再現
- **コマンドライン引数**: `$1`, `$2`などのbashの変数と`os.Args`の対応を理解
- **エラーハンドリング**: bashの`if [ $? -ne 0 ]`とGoの`if err != nil`の違いを学ぶ
- **並行処理**: bashのバックグラウンド実行（`&`）とGoのゴルーチンの違いを理解

## 注意事項

- このリポジトリは学習用であり、プロダクション環境での使用を想定していません
- 各レベルのコードは学習過程の記録であり、最適化されていない場合があります
